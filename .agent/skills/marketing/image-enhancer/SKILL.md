# Image Enhancer

## When to Use This Skill
- Improving screenshot quality for documentation.
- Enhancing images before sharing.
- Preparing images for presentations or reports.
- Upscaling low-resolution images.
- Sharpening blurry photos.
- Cleaning up compressed images.

## What This Skill Does
1.  **Analyzes Image Quality**: Checks resolution, sharpness, and artifacts.
2.  **Enhances Resolution**: Upscales images intelligently.
3.  **Improves Sharpness**: Enhances edges and details.
4.  **Reduces Artifacts**: Cleans up noise.
5.  **Optimizes for Use Case**: Adjusts based on intended use (web, social).

## How to Use
- **Basic Enhancement**: "Improve the image quality of [filename]"
- **Specific Improvements**: "Sharpen text and upscale [filename]"
- **Batch Processing**: "Enhance all images in the folder"

## Tips
- Always keep original files as backup.
- Works best with screenshots and digital images.
- Can batch process entire folders.
